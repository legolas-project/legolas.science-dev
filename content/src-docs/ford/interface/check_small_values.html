private subroutine small_values_real_array(array, tol)

Small value checks on a real array. Checks a real double-precision array for small values. Values that are smaller than the specified tolerance tol are set to zero. If tol is not present, DP_LIMIT is used as tolerance.

Arguments

Type IntentOptional AttributesName
real(kind=dp), intent(inout) :: array(:)

the real array to check
real(kind=dp), intent(in), optional :: tol

the tolerance to check against

private subroutine small_values_complex_array(array, tol)

Small value checks on a complex array. Checks a complex double-precision array for small values. Values that are smaller than the specified tolerance tol are set to zero. The real and imaginary parts are checked (and set) separately. If tol is not present, DP_LIMIT is used as tolerance.
