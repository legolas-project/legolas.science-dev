.. py:class:: IVPSolution(times, data, component_names=None, x_domain=None, units=None)

   
   Container for IVP snapshot data returned by Legolas.

   :param times: Physical times at each snapshot, shape ``(n_snap,)``.
   :type times: np.ndarray
   :param data: Snapshot data, shape ``(n_snap, n_comp, n_points)``.
   :type data: np.ndarray
   :param component_names: Map from integer index to component name, e.g. ``{0: "rho", 1: "v1"}``.
   :type component_names: dict, optional
   :param x_domain: Spatial coordinates, shape ``(n_points,)``.
   :type x_domain: np.ndarray, optional
   :param units: Unit normalisations from the Legolas datfile header.
   :type units: dict, optional
